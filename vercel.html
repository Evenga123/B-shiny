Ši nuoroda importuoja jau egzistuojantį GitHub repo. Ji nebando kurti naujo repo, todėl neturėtų rodyti klaidos, kad vardas jau egzistuoja.

  1. Spausk mygtuką.
  2. Vercel puslapyje rinkis Import esamam repo.
  3. Jei matai Create Repository, tai ne tas kelias - grįžk atgal ir importuok esamą repo Evenga123/B-shiny.
  4. Framework Preset palik Other, Build Command palik tuščią, Output Directory palik ..
